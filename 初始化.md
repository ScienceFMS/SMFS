1. **克隆项目**：
   
   ```
   git clone [项目仓库URL]
   cd [项目文件夹]
   ```
   
2. **后端设置**：
   ```
   cd BackEnd/scienceFMS
   ```
   - 确保安装JDK 17
   
   - 确保Maven配置正确
   
   - 运行 `mvn clean install`

     
   
3. **前端设置**：
   ```
   cd FrontEnd/[前端项目文件夹]
   npm install
   npm run dev
   ```

4. **开发规范**：
   
   - 从master创建新功能分支
   - 完成后提交PR
   - 遵循代码规范和提交信息规范
